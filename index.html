What is CurveSwap?

CurveSwap is an app-first destination for users looking for Curve's token swap flow, with the primary action routed to the official Curve DEX. Use the Curve documentation for protocol background before signing transactions.

CurveSwap FAQ

What is curveswap?

curveswap is a search term for users looking for Curve's swap experience; this page routes to the official Curve DEX and keeps educational context separate from the app.

Is curveswap the official Curve app?

No. This landing page is not the official app; use www.curve.finance/dex/ for the Curve DEX and verify the domain.

Does curveswap set fees or APY?

No. Swap fees, routes, price impact, and available pools are shown by the live app and can change by network, pool, and market conditions.

Which wallet should I use with curveswap?

Use a self-custody wallet you control and review Ethereum wallet basics at ethereum.org before signing transactions.

Can I swap stablecoins with curveswap?

Curve is known for liquidity pools used by stable and pegged assets, but always check the available pool, route, and token contracts in the app before trading.
